# Helpdesk Ticket Template Generator

A PowerShell tool that generates structured helpdesk ticket notes.

## Features

- Incident template
- Service request template
- Escalation template
- User callback template
- Markdown output
- HTML preview output

## How to run

```powershell
powershell.exe -ExecutionPolicy Bypass -File .\Helpdesk_Ticket_Template_Generator.ps1
```

## Safety

This script generates documentation only.
